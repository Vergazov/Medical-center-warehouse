## Таблица Номенклатура - nomenclatures

> Хранит информацию о номенклатуре медциниского центра

| Поле               | Тип данных    | Описание                          |
|--------------------|---------------|-----------------------------------|
| id                 | INTEGER       | Уникальный Идентификатор          |
| name               | VARCHAR       | Название                          |
| type_id            | INTEGER       | Тип                               |
| speciality_id      | VARCHAR       | Специальность                     |  
| main_unit_id       | VARCHAR       | Основная единица измерения        |

| expiration_date    | VARCHAR       | Срок годности                     |
| expiration_unit_id | VARCHAR       | Единица измерения срока годности  |
| min_balance        | SAMLLINTEGER  | Максимальный остаток              |
| max_balance        | SAMLLINTEGER  | Минимальный остаток               |
| comment            | VARCHAR       | Комментарий                       |
